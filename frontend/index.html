❓ Help & Tutorial

🚀 Getting Started

  1. Ensure LM Studio is running with your preferred model loaded
  2. Start the backend server: uvicorn app.main:app --port 8003
  3. Select a voice personality from the left panel
  4. Use "Hold To Talk" for voice input or type in the text box

⚙️ Settings

  • Backend URL: Address of your AIVoice backend server
  • LLM API URL: LM Studio API endpoint (usually ends with /v1/chat/completions)
  • Voice Model Paths: Directories containing Piper TTS and Whisper models

🎤 Voice Features

  • Multiple voice personalities available
  • Dual-response system (spoken brief + detailed text)
  • Real-time speech-to-text transcription
  • Text-to-speech playback

💡 Tips

  • Click the gear icon (⚙️) to access full settings
  • Use the sun/moon icon (☀️) to toggle light/dark mode
  • Individual copy buttons appear on code blocks and responses
  • Voice panel can be minimized by clicking the arrow
